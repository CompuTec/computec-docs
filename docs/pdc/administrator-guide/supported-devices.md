# Supported Devices

CompuTec PDC runs on devices that support the **SAP Business One DI API**. This includes a wide range of standard and industrial hardware.

You can use CompuTec PDC on:

- Desktop PCs
- Industrial computers
- Windows tablets

The key requirement is that the device can reliably run SAP Business One components.

## Supported Operating Systems

CompuTec PDC is supported on the following Windows versions:

- Windows 7
- Windows 8
- Windows 10
- Windows 11

:::caution[important]
CompuTec PDC is not supported on devices running Windows RT.
:::

## Recommended Devices

For the best experience, use hardware that has been tested in real warehouse environments with CompuTec PDC.

### **DURABOOK R11 tablet**

The DURABOOK R11 is a device well-suited for warehouse use:

- Display: 11.6" FHD (1920 x 1080) Display Standard with Touch
- Processor: Intel i5-5200U
- Memory: 4GB
- Storage: 128GB MLC SSD
- Connectivity: WLAN+BT Intel 7260
- Cameras: 2M Front Camera; 5M Rear Camera
- Ports: 2x USB 3.0, SDHC
- Other Features: Dock Connector with Pass-Through, TPM 1.2, AC Adapter Power Cord, and a Standard 4130mAh Battery
- Durability: IP65 certified, MIL-STD-810G/F compliant

This type of device is recommended for environments where durability, mobility, and reliable connectivity are important.

## Best Practice

- Use **industrial or rugged tablets** in warehouse environments
- Avoid low-spec devices that may slow down SAP communication
- Always verify compatibility with **SAP Business One DI API**
