# Overview

The CompuTec Manufacturing Cockpit (MO360) plugin is designed to monitor the progress of Manufacturing Orders in real time. It features a planned Gantt chart that visualizes the workflow and highlights ongoing activities. Additionally, the plugin includes an Assignment Board, enabling the assignment of employees and teams to specific tasks.
