# CompuTec Manufacturing Cockpit (MO360) Plugin

Below are the release notes for CompuTec Manufacturing Cockpit (MO360) Plugin, highlighting new features, improvements, and bug fixes in this release.

## CompuTec MO360 3.2602.1

**Release Date: 27 February 2026**

| Issue Type | Components | Release Note |
| --- | --- | --- |
| New | MO360 | Added **Material Shortages** report. This feature (available via Analytics Actions) allows users to analyze material shortages both for single and across multiple **Manufacturing Orders** in one consolidated report. |
| Fixed | AE Configuration | Fixed an issue where opening **Additional Settings** for plugins could display an incorrect ``configuration not found`` message. The **Additional Settings** dialog now opens correctly for plugins that support it. |

## CompuTec MO360 3.2508.1

**Release Date: 04 August 2025**

| Issue Type | Components | Release Note |
| --- | --- | --- |
| Fixed | MO360 | Fixed an issue where the **Display Details** section showed no data even when records were available. Details now load correctly and consistently. |
| Fixed | MO360 | Fixed an issue that prevented users from creating new views. View creation now works as expected. |
