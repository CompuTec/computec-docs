# Log Files

The CompuTec Premium logs are found in the directory C:\ProgramData\CompuTec Label Printing\Logs. Within this directory, the logs are categorized as follows:

- **CTLP** - service logs,
- **CTLP** - manager logs,
- **WixSetup** - installation logs.
