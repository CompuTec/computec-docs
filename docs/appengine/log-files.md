# Log Files

If the **CompuTec AppEngine** installation fails, you can check system logs to identify the issue.

## Windows Event Viewer

Installation errors are recorded in the **Windows Event Viewer**. Review these logs first to find detailed error messages.

## Application Logs

**CompuTec AppEngine** also writes its own log files to the following location:

```http
C:\ProgramData\CompuTec\AppEngine\logs
```

These logs provide additional details that may not appear in the **Windows Event Viewer** and can help with troubleshooting.
