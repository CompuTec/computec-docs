# Plugin Update

You can update plugins using the **Releases** tab in the **CompuTec AppEngine Administration Panel**.

![go to releases section](./media/update/plugin-update-1.png)

## Update a plugin

To update an installed plugin, follow these steps:

1. Log in to **CompuTec AppEngine Administration Panel**.

    ![log into computec appengine administration panel](./media/installation/plugin-install1.png)

2. Go to **Plugins**.

    ![go to plugins section](./media/installation/plugin-install2.png)

3. Navigate to **Releases**.

    ![go to releases section](./media/update/plugin-update-1.png)

4. Click the **Get...** to install the latest plugin version.

    ![click the number of the version](./media/update/plugin-update-2.png)

5. (optional) If you want to install a different version, foolow these steps:

    - Click the **plugin name**.

        ![click the plugin name to find a different version](./media/update/plugin-update-3.png)

    - Navigate to **Other Versions**.

        ![click other versions](./media/update/plugin-update-4.png)

    - Choose the **version** you want to install.

        ![click the version number](./media/update/plugin-update-5.png)

6. Click **Get**.

    ![click get](./media/update/plugin-update-6.png)

7. Click **Get & Install**.

    ![click get & install](./media/update/plugin-update-7.png)

8. Select the company for installation and click **Accept**.

    ![select the company and click accept](./media/update/plugin-update-8.png)

9. Select **CompuTec AppEngine Instance** for installation and click **Accept**.
 
    ![select the instance and click accept](./media/update/plugin-update-9.png)

10. Review the installation details and click **Perform Installation**.

    ![click perform installation](./media/update/plugin-update-10.png)

11. Click **OK**.

    ![click ok](./media/update/plugin-update-11.png)

12. After the installation is completed, click **Close**.

    ![click close after the installation is completed](./media/update/plugin-update-12.png)

13. Click **Yes** to restart **CompuTec AppEngine**.

    ![click yes to restart computec appengine](./media/update/plugin-update-13.png)

14. After the restart is completed, click **Close**.

    ![after restart of computec appengine, click close](./media/update/plugin-update-14.png)

15. Done! You’ve successfully updated the plugin.

## After update

After the successful update:

- The plugin runs in the selected version
- The previous version is replaced on the Computec AppEngine instance

:::info[Note]

To learn about new features or changes, refer to the plugin’s [documentation](http://learn.computec.one/docs/appengine/plugins-user-guide/overview#available-plugins).

:::
