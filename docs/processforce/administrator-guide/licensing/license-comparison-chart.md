# License Comparison Chart

This chart shows which **CompuTec ProcessForce** features are available for each **CompuTec ProcessForce** license type.

Use it to compare license levels and verify which functionality is available to your users.

## License compatibility

A user's **CompuTec ProcessForce** license must be equivalent to or higher than their **SAP Business One** license type.

For example:

- A user with an **SAP Business One Limited Logistics** license can be assigned a **CompuTec ProcessForce Limited Logistics** license.
- A user with an **SAP Business One Limited Logistics** license can also be assigned a **CompuTec ProcessForce Professional** license.
- A user cannot be assigned a **CompuTec ProcessForce** license that provides a lower access level than their **SAP Business One** license.

Refer to the comparison chart below to determine the appropriate **CompuTec ProcessForce** license for each **SAP Business One** user type.
