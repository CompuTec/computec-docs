# CompuTec Batch Traceability (BT 360) Plugin

Below are the release notes for CompuTec Batch Traceability (BT 360) Plugin, highlighting new features, improvements, and bug fixes in this release.

## CompuTec BT360 3.2604.1

**Release Date: 10 April 2026**

| Issue Type | Components | Release Note |
| --- | --- | --- |
| Improved | Web API | Removed unused API calls, improving system performance and reducing unnecessary background operations. |
| New | BT360 | Introduced the ability to create and manage custom **Views**. You can now filter data, save the filtered view, and set it as **Default**, **Private**, or **Public**, giving you more control over how data is displayed. |
| Improved | BT360 | Added the ability to filter data easier and create new filtered views in **Trace Analysis** > **Links**, enabling faster access to relevant information. |
| Fixed | BT360 | Fixed an issue where opening a batch could trigger the error ``Unable to cast object of type 'System.DBNull' to type 'System.Decimal'``. Data now loads without errors. |
