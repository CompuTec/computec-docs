# Video Tutorials

Here you'll find all our videos about **CompuTec WebUp**. These short guides will help you learn how CompuTec WebUp works, how to set it up, and how to use its key features.

All videos link directly to [**CompuTec YouTube channel**](https://www.youtube.com/@computecvideo) and open in a new tab.

## Learn the basics

These videos introduce the basics of CompuTec WebUp and help you begin working with the app.

- [▶ **CompuTec WebUp - First Launch**](https://youtu.be/YO3jdjChS4k): In this video, you’ll see how to launch CompuTec WebUp for the first time.

## Explore key features

Here you’ll find videos that explain specific CompuTec WebUp features and show how to use them in everyday work.

- [▶ **CompuTec WebUp - How to Hide Controls**](https://www.youtube.com/watch?v=HBFxPcV0ris): Learn how to hide controls in the SAP Business One Web Client using CompuTec WebUp.

- [▶ **CompuTec WebUp - How to Move Controls**](https://www.youtube.com/watch?v=bUEkcdizCLY): Learn how to reposition controls within the SAP Business One Web Client.

- [▶ **CompuTec WebUp - Controls - Other Configurations**](https://www.youtube.com/watch?v=MQk_z-cr-v8): Discover additional configuration options for controls, including setting fields as mandatory or read-only, changing labels, defining input types, adding tooltips, and more.

- [▶ **CompuTec WebUp - Validation - Creating a Warning Prompt**](https://www.youtube.com/watch?v=2CcmJlgOAb8): See how to create a validation rule that displays a warning message when a required field is left empty.

- [▶ **CompuTec WebUp - Automation - Creating Activities and Actions**](https://www.youtube.com/watch?v=VYICWbt62YY): Learn how to create automations, including an example where the system automatically generates a follow-up activity based on specific conditions.
