# CompuTec WebUp Manager Release Notes

## CompuTec WebUp Manager 1.0.15

**Release Date: 13 April 2026**

| Type | Release Note |
| --- | --- |
| Fixed | Improved **Content Security Policy (CSP)** handling for **SAP Business One Web Client** environments. Fixed an issue where **CSP** details were not available on servers using trusted certificates, which could prevent **CompuTec WebUp Manager** from loading correctly. This issue no longer requires refreshing the page or reopening Computec WebUp Manager. |
| Improved | Redesigned the **Preview** mode banner to make it clearer and easier to notice. |
